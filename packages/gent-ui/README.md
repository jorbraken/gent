# Gent UI

Visual source-aware editor for Gent profiles, sandboxes, context, skills, memories, decisions, and pipelines, inside VS Code.

## Features

- Browse and edit Gent entities (profiles, sandboxes, context, skills, memories, decisions, pipelines) from the activity bar.
- Inspect and preview profiles.
- Manage sandboxes: validate, run, view logs, stop, and destroy.

## Requirements

Requires a Gent-managed project (see the root [gent](https://github.com/jorbraken/gent) CLI) to detect entities.
